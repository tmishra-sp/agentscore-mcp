## AgentScore MCP v1.0.1

This release hardens public-facing reliability and release quality.

### Highlights
- Added adapter contract regression tests for GitHub and Moltbook retry/outage behavior.
- Added CI coverage gating using `c8` with enforced minimum thresholds.
- Tightened CI checks to enforce build, typecheck, full test suite, and coverage.
- Improved sweep behavior with content-only fallback when participant profiles are unavailable.
- Improved GitHub rate-limit handling for both `403` and `429` responses.
- Added fast-fail validation for invalid `AGENTSCORE_ADAPTER` configuration.
- Removed shared anonymous rate-limiting collisions by using resolved stable client keys.

### Validation
- `npm test` passing.
- `npm run coverage` passing with enforced thresholds.
