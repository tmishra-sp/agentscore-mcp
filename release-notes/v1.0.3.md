## AgentScore MCP v1.0.3

This release aligns public report links with live site routes, adds release automation, and tightens public-facing contracts.

### Highlights
- Switched MCP `reportUrl` format to handle-based routes: `/agent/{handle}`.
- Added a tag-driven npm release workflow (`v*` tags) with:
  - tag/package version consistency check
  - full `npm run verify` gate
  - `npm publish --access public`
- Added public contract regression tests for:
  - README install command/package name consistency
  - README npm links consistency
  - report URL route shape
- Added JSON sample dataset reference used in README.
- Added launch kit assets for distribution execution.

### Validation
- `npm test` passing.
- `npm run coverage` passing.
- `npm run benchmark:strict` passing.
